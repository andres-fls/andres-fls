Modern web-based event management system developed for SENA.

Built with **ASP.NET MVC, C#, SQL Server, JavaScript, HTML, CSS, Azure App Service, and Azure SQL Database**.

Includes user authentication, role-based authorization, event management, individual and group registrations, client and server-side validations, AI chatbot integration, and cloud deployment.

### 🗓️ EventPlus

Desktop event management system built for **SENA academic projects**.

Built with **C# Windows Forms + MongoDB Atlas** — includes role-based authentication (Leader/Invitee), event CRUD, attendee management with schedule conflict validation, and monthly filtering.

→ [View repository](https://github.com/andres-fls/eventPlus)
